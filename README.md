# Subway Surfers: Kid on the Run

A fun pygame-based endless runner game where you control a kid running away from a cop in the subway!

## How to Play

### Controls
- **LEFT ARROW or A**: Move left
- **RIGHT ARROW or D**: Move right
- **C**: Restart the game (when game is over)

### Objective
- Avoid obstacles (brown boxes) falling from the top
- Don't get caught by the cop (blue character) chasing from the bottom
- Survive as long as you can to increase your score
- The game gets harder as you progress (faster obstacles and cop)

### Game Mechanics
- **Player**: You control a yellow character (the kid) in the middle of the screen
- **Cop**: A blue character that chases you from the bottom, moving upward
- **Obstacles**: Brown boxes that fall from the top
- **Score**: Based on the time you survive
- **Cop Speed Boost**: If you hit an obstacle, the cop speeds up!
- **Difficulty**: Increases as time passes (obstacles spawn faster, cop moves faster)

## Installation

1. Make sure you have Python installed
2. Install pygame:
   ```
   pip install -r requirements.txt
   ```

## Running the Game

```
python main.py
```

## Tips
- Move left and right to dodge falling obstacles
- Keep an eye on the cop advancing from below
- Hitting obstacles makes the cop faster - be careful!
- The longer you survive, the more difficult it gets!

Have fun running! 🏃‍♂️💨
